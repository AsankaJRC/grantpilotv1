Construction rules

Set in Momo Trust Sans weight 800–900. "Funding" renders in the surface-appropriate ink colour. "Scribe" is always #236cff — never any other colour. Letter-spacing -0.4px. Do not use Display weight for the wordmark. Never rotate, skew, recolour, or separate the two words.

07
Brand rules
Do
✓

Always render "Scribe" in brand blue #236cff

✓

Use gradient text only on hero H1 and major headings

✓

Set eyebrows in UPPERCASE with 1.2–1.4px letter-spacing

✓

Use Display font for all headings and large numerals

✓

Pair stat numbers with muted 14px descriptor text

✓

Apply frosted glass to nav on scroll (blur + opacity)

✓

Use grid backgrounds in hero sections at 0.06–0.1 opacity

✓

Card hover: translateY(-3px) + blue border tint

✓

Reference ACNC / DGR / SmartyGrants in compliance copy

✓

Specify AWS ap-southeast-2 for all security contexts

Don't
✕

Render "Funding" in blue — ink colour only

✕

Apply gradient to body or supporting text

✕

Reuse the animated hot-gradient outside the featured plan

✕

Use border-radius below 8px on any UI element

✕

Rotate, skew, or distort the wordmark

✕

Apply drop-shadows to cards (hover uses border-colour change)

✕

Use #00fff5 cyan anywhere except the featured plan dot

✕

Drop font size below 11px anywhere in the UI

✕

Spell out numbers in stats — always use numerals

✕

Use dark backgrounds without strong design purpose
